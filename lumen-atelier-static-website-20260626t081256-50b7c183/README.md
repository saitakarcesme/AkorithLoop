# Lumen Atelier Static Website

A one-loop Akorith project: Claude should build a real responsive static website, not only planning markdown.

## Loop Goal

Create an English, polished, responsive HTML/CSS website for a fictional creative studio called Lumen Atelier. The project must include real website files (`index.html`, `styles.css`) and enough content/styling to open directly in a browser.

## Run notes

No build step required. Open directly in any browser:

- `index.html` — full single-page site (hero, services, work, process, testimonial, contact, footer)
- `styles.css` — all layout and visual styles; no external assets or JavaScript
