# Repository health analysis loop

This workspace was created for an Akorith fully autonomous Loop.

## Goal
Analyze repository health every week: stale code, weak README sections, missing tests, missing CI, outdated dependencies, bad folder structure, UX gaps, and improvement opportunities. Produce prioritized issues/tasks and a health report.

## Safety
- Do not expose secrets.
- Do not run destructive commands without approval.
- Report validation honestly.
