# Pull request review and release prep loop

This workspace was created for an Akorith fully autonomous Loop.

## Goal
Every 3 hours, review open PR/release signals, summarize risks, suggest review comments, prepare changelog/release notes, identify blockers, and create a clear next-action report without publishing externally unless approved.

## Safety
- Do not expose secrets.
- Do not run destructive commands without approval.
- Report validation honestly.
